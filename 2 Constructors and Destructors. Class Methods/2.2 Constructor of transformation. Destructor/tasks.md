## Задача 3 - Point (конструктор преобразования из строки)

Дан класс `Point`:

```cpp
class Point {
public:
    int get_x() { 
        return x; 
    }

    int get_y() { 
        return y; 
    }

private:
    int x, y;
};
```

Добавьте в этот класс конструктор(ы) так, чтобы были возможны следующие команды:

```cpp
Point p1 = "10 5";                   // x = 10, y = 5
Point p2 = "-8.7 2.3";               // x = -8, y = 2
Point p3 = "14.2 -5.8 point data";   // x = 14, y = -5
```

Из строки выбираются первые два значения, записанные через пробел(ы), и приводятся к типу `int`. Если данные оказываются не числовыми, подставляется значение `0`. Например:

```cpp
Point p_1 = "ab8 2.3";  // x = 0, y = 2
Point p_2 = "ab8 2cd";  // x = 0, y = 2 (cd отбрасывается)
```

## Задача 4 - Vector3D (конструктор преобразования из строки и массива)

Дан класс `Vector3D`:

```cpp
class Vector3D {
public:
    void get_coords(int& a, int& b, int& c) {
        a = x; 
        b = y; 
        c = z;
    }

private:
    int x, y, z;
};
```

Добавьте в этот класс конструкторы так, чтобы были возможны следующие команды:

```cpp
Vector3D v1 = "1 2 -5";               // x = 1, y = 2, z = -5
Vector3D v2 = "1.5 2.2 -5.7";         // x = 1, y = 2, z = -5
Vector3D v3 = "1.5 2.2 -5.7 data";    // x = 1, y = 2, z = -5

int coords_1[] = {2, 4, 7}, coords_2[] = {5, 3, 1, 0, 20};
Vector3D v4 = coords_1;               // x = 2, y = 4, z = 7
Vector3D v5 = coords_2;               // x = 5, y = 3, z = 1
```

На вход может поступать строка (`const char*`) или целочисленный массив (`int*`). Из строки выбираются первые три значения через пробел(ы), приводятся к типу `int`. Нечисловые данные заменяются на `0`. Из массива выбираются первые три числа.

## Задача 7 - IntOperator (деструктор)

Объявите класс с именем `IntOperator`, содержащий следующие элементы:

**Приватные переменные:**

```cpp
int* data{nullptr};  // массив целочисленных значений
int size{0};         // длина массива (число элементов)
```

**Публичные методы:**

```cpp
double average();  // вычисление среднего арифметического по массиву data
int sum();         // вычисление суммы значений массива data
int max();         // нахождение максимального значения в массиве data
int min();         // нахождение минимального значения в массиве data
```

В классе `IntOperator` объявить конструктор(ы), чтобы его объекты можно было создавать командами:

```cpp
int dt[] = {0, -5, 2, 10, 7, 22};

IntOperator op_1;
IntOperator op2(dt, std::size(dt));
```

Объявите в классе `IntOperator` деструктор, чтобы при удалении объекта значения переданного массива (в примере `dt`) были равны нулю.

## Задача 8 - VectorN (деструктор с выводом)

Объявите класс с именем `VectorN`, содержащий следующие элементы:

**Приватные переменные:**

```cpp
int dims{0};            // размерность вектора (число координат)
short* coords{nullptr}; // указывает на массив с координатами
```

**Публичные методы:**

```cpp
const short* get_coords();   // возвращает координаты (указатель coords)
int get_dims();              // возвращает значение поля dims
```

Указатель `coords` должен вести на выделенную область памяти, полученную с помощью оператора `new`.

В классе `VectorN` объявить конструктор(ы) так, чтобы его объекты можно было создавать командами:

```cpp
short data[] = {4, 2, 10, 0, -5};
VectorN v1 = 5;                       // вектор с пятью нулевыми координатами: 0 0 0 0 0
VectorN v2(1, 2);                     // вектор с двумя координатами: 1 2
VectorN v3(data, std::size(data));    // вектор с переданными координатами data
```

При удалении объекта нужно освободить память, занимаемую массивом `coords`, и вывести в консоль сообщение (с переносом на новую строку):

```
delete coords: <координаты через пробел>
```

В конце строки пробела быть не должно. Если `coords` равен `nullptr`, ничего выводить не нужно.

Пример вывода при `coords = {4, 2, 10, 0, -5}`:

```
delete coords: 4 2 10 0 -5
```
